Text Editing


The link to the code is here!

Objectives of this tutorial:

1. Learn to edit text
2. Learn to display text from a variable

Exercises:

1. When you run the code in this example, you can see “Hello, World” displayed on the screen. Find “Hello, World!” in the code, and change it to say “Hello [your name]!”.

Example:
My name is Jon, so it says “Hello, Jon!”.


2. Variables allow us to save things that we can use later. Use a variable to store “Hello, [your name]!”, and display the variable to the screen.

Example:
We can save text to a variable like this:
ourText = “This is text”
We show a variable like this:
show ourText
